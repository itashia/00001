نمونه کار وب سایت آزمونی

قابلیت ساخت آزمون های چندگزینه ای با تعداد نامحدود سوالات
امکان ساخت سوالات باز (free-text) برای پاسخ دهی شرکت کنندگان
امکان اضافه کردن تصاویر و فایل های چندرسانه ای به سوالات
امکان تعیین نمره دهی برای هر سوال و آزمون
امکان تنظیم ترتیب یا تصادفی بودن سوالات در آزمون
قابلیت افزودن توضیحات و راهنما برای هر سوال
قابلیت ایجاد بانک سوالات و مدیریت آنها
امکان تعیین سطح سوالات (مثلاً ساده، متوسط، سخت)
امکان اضافه کردن بخش ها و زیربخش های آزمون
قابلیت ارسال پیام و اعلان به شرکت کنندگان آزمون
امکان استفاده از کد‌های محافظتی (کپچا) در آزمون
قابلیت تنظیم محدودیت تعداد دفعات شرکت در آزمون
امکان افزودن نام و لوگوی سازمان یا آموزشگاه به آزمون
امکان تنظیم پاسخ تشریحی برای سوالات
قابلیت ویرایش آزمون ها پس از ایجاد آنها
امکان تعیین نمره‌ی منفی برای پاسخ های اشتباه
قابلیت ارائه آزمون به صورت چندگانه در یک زمان
امکان تنظیم سوالات به صورت تصادفی از بانک سوالات
قابلیت استفاده از فایل های صوتی در سوالات
امکان ایجاد آزمون های تشریحی و ارزیابی نوشتاری
امکان ایمپورت سوالات از فایل خارجی
امکان ارائه گواهی و کارنامه
امکان تغییر قالب گواهینامه ها
و بسیاری امکانات دیگر
